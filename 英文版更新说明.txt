HOW TO RUN:
========================================

Option 1 - Quick Start (Recommended):
   Double-click: "启动完整项目.bat"
   This will start both API and client automatically!

   Step 1: Double-click "启动API服务器.bat"

   Step 3: Open http://localhost:8080/index.html

   Terminal 1: cd api && npm start
   Terminal 2: cd client && python -m http.server 8080
